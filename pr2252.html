⚠️ Golden Rules (4)

🔴 1. NEVER use fetch() for internal API calls

Use Server Actions instead. Fetching your own API adds latency and bypasses type safety.

🔴 2. DB queries stay in lib/db/queries/

One file per query group. No inline SQL in Components or Actions.

🔴 3. Every mutation has a loading state

Server Actions return discriminated unions with useActionState wrapper.

🔴 4. Migrations are immutable once merged

Never modify a merged migration — create a new one to preserve audit trail.
